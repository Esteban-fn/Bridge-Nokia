Ajuda - Sistema de Bridge

O que é Bridge?

O Bridge em redes de fibra (FTTH) é a configuração que conecta a ONT diretamente à rede do cliente, sem que a ONT funcione como roteador. Dessa forma, o roteador do cliente recebe o IP público e autentica diretamente via PPPoE.

Para que serve?

  • Permitir que o cliente use o próprio roteador para autenticar a conexão.
  • Entregar IP público diretamente ao roteador do cliente.
  • Reduzir problemas de NAT duplo (quando há dois roteadores em cascata).

Em quais situações é feito?

  • Clientes que possuem roteadores próprios (Mikrotik, TP-Link, Intelbras, etc.).
  • Quando o cliente precisa de controle avançado da rede.
  • Em contratos corporativos que exigem IP fixo público.

Prós e Contras

Prós

  • Mais controle para o cliente.
  • Entrega de IP público diretamente.
  • Evita NAT duplo.

Contras

  • Dependência do roteador do cliente para autenticar.
  • Se o cliente não configurar corretamente, pode ficar sem internet.

Como fazer manualmente (linha por linha)

Exemplo de configuração:

configure equipment ont interface 1/1/SLOT/PON/POS sw-ver-pland auto desc1 DESC1 desc2 DESC2 sernum ALCL:XXXXXXX subslocid WILDCARD fec-up disable optics-hist enable sw-dnload-version disabled voip-allowed veip log-auth-pwd plain:** pland-cfgfile1 auto dnload-cfgfile1 auto planned-us-rate nominal-line-rate

configure equipment ont interface 1/1/SLOT/PON/POS admin-state up

configure equipment ont slot 1/1/SLOT/PON/POS/CARD planned-card-type ethernet plndnumdataports 4 plndnumvoiceports 0 admin-state up

configure qos interface 1/1/SLOT/PON/POS/CARD/PORTA upstream-queue 0 bandwidth-profile name:HSI_1G_UP 

configure interface port uni:1/1/SLOT/PON/POS/CARD/PORTA admin-up
configure bridge port 1/1/SLOT/PON/POS/CARD/PORTA max-unicast-mac 12 max-committed-mac 1
configure bridge port 1/1/SLOT/PON/POS/CARD/PORTA vlan-id VLAN
configure bridge port 1/1/SLOT/PON/POS/CARD/PORTA pvid VLAN

Explicação linha a linha

  • configure equipment ont interface ...: Cria a interface da ONT com número de série e descrições.
  • admin-state up: Sobe a ONT.
  • configure equipment ont slot ...: Cria a placa de rede na ONT.
  • configure qos interface ...: Configura fila de upstream e perfil de banda.
  • configure interface port uni ...: Habilita a porta física.
  • configure bridge port ... vlan-id: Define a VLAN de serviço.
  • configure bridge port ... pvid: Define VLAN padrão da porta.

Como fazer pelo sistema

  1. Selecione o tipo de Bridge (Rede ou VoIP).
  2. Escolha a cidade e a VLAN será preenchida automaticamente.
  3. Preencha Slot, PON, Posição e S/N (ALCL).
  4. Digite a descrição (Desc1 e Desc2) conforme padrão.
  5. Escolha a porta LAN que será usada.
  6. Clique em Gerar Comandos.
  7. Copie e cole no AMS os comandos gerados.
